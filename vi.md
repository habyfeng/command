
### vi定位

```
H - 移至屏幕上端
M - 移至屏幕中央
L - 移至屏幕下端
```

### 设置行号
```
:set nu    
```

### 取消行号
```
:set nonu
```

### 到第一行
```
gg
```

### 到最后一行
```
G
```

### 到第n行
```
nG
```

### 到第n行
```
:n
```

### 删除光标所在行
```
dd
```

### 删除光标所在行到文件最后一行的内容
```
dG
```

### 删除光标所在处到行尾内容
```
D
```

### 删除n1行到n2行
```
:n1,n2d
```

### 撤销
```
u
```

### 回复被撤销的操作
```
Ctrl + r
```
